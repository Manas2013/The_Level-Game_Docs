About the Game

The Level is a game designed by Manas Pankaj Jawale. This is a game where you are protagnist and A man named Eric is your guardian is this entire adventure.
By the name, this game has some main missions:

  • Complete the Levels coming
  • Increase your speed in parkour
  • Complete the new levels coming in each update

Keys to be used

Because this game is in development and does not have really much system, this game is still in it's alpha stage. But there are some features I've made in it yet:

Basic Controls
  • WASD to move
  • Arr keys to move
  • Mouse to rotate view
  • CTRL to crouch
  • ESC to toggle mouse
  • Shilf to Sprint
Addictional Controls
  • RMB to zoom in
  • R to Respawn
